Hooks

Deterministic. Not AI. Shell scripts that fire on agent events.

Hooks turn vibes into rules. Git hooks, but for your agent.
Matcher · pattern on tool call
Bash(rm *)
Wildcard matchers on tool name

echo to stdout = inject context
Append to file = audit log
All hook types

PreToolUse.sh — block dangerous commands
 hooks/PreToolUse.sh
#!/bin/bash # exit 2 = block | exit 0 = allow TOOL="$1"   INPUT=$(cat) if [[ "$TOOL" == "bash" ]]; then CMD=$(echo "$INPUT" | jq -r '.command') if echo "$CMD" | grep -qE 'rm\s+-rf'; then echo "BLOCKED: rm -rf not allowed" >&2; exit 2 fi if echo "$CMD" | grep -q 'kubectl.*prod'; then if ! echo "$CMD" | grep -q '--dry-run'; then echo "BLOCKED: prod kubectl requires --dry-run" >&2; exit 2 fi fi fi exit 0
PostToolUse.sh — auto-lint + Slack notify
 hooks/PostToolUse.sh
#!/bin/bash FILE=$(echo "$(cat)" | jq -r '.path // empty') if [[ "$1" == "write_file" ]] && [[ "$FILE" == *.py ]]; then ruff check --fix "$FILE" 2>/dev/null fi exit 0
